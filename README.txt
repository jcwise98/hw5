An environment of primarily 3d objects
- The board and the player pieces are 3d models.

User control through keyboard or mouse input
- The user can control where they place a piece by using the numbers 1-7 on their keyboard coresponding to the column they want to place the piece.

Objects must move smoothly
- An animation moving the player piece from the top of the screen is used via linear interpolation.

At least one texture-mapped element
- The board and player pieces are texture mapped elements

At least one 3-d element which is lit with diffuse and/or specular lighting
- The board is has lighting

A scoreboard or other text-based overlay
- A text overlay for the status of the game (whose turn it is and or who won the game) is used

-Complicated 3-d objects
- The game board is a complicated 3d object
